# TaiwanTravel
以 Vue2 實作台灣旅遊資訊網

## Routes
```
首頁
|-- /

搜尋頁
|-- /:category/:city/:q?page

資訊頁
|-- /information/:category/:city/:id
```
## 功能
* 首頁隨機任選呈現台灣旅遊資訊
* 任選 *縣市區域 、 * 資訊種類 呈現其搜尋結果
* 詳細資訊頁面顯示其周邊的其他資訊
* 支援 RWD

## 學習運用

* 串接 API，連接整理資料
* Flex 排版
* 圖片上一層濾鏡以免呈現雜亂
